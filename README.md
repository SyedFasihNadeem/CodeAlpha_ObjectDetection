## Project Description
A real-time object detection system that uses your webcam to detect and track 80 different object classes in real-time, running entirely in the browser using TensorFlow.js and the COCO-SSD pre-trained model.

## Features
- Real-time webcam object detection
- 80 object classes from the COCO dataset (person, car, laptop, cup, dog, cat, etc.)
- Bounding boxes drawn live on detected objects
- Confidence scores for each detection
- Live FPS counter
- Object count tracking
- 100% privacy — all detection happens locally in the browser, no server upload

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- TensorFlow.js
- COCO-SSD pre-trained model
- HTML5 Canvas for bounding box rendering
- WebRTC getUserMedia for webcam access
